### 深度学习入门 - 基于Python的理论与实现

本项目基于Python和Numpy，对深度学习（神经网络）进行了初步的介绍，每个文件夹下均附有读书笔记`Brief.pdf`

**参考书**：《深度学习入门：基于Python的理论与实现》（斋藤康毅）

**项目结构：**

|       ...       |    ActivationFunction.py    |        激活函数        |
